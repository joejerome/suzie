    Sefray entreprise

    Sefray Entreprise est une institution créée en juin 2019 dans la commune de carrefour par un jeune entrepreneur œuvrant dans le domaine de l’éducation. Après plusieurs séances de formations suivies sur la chimie industrielle, ce jeune leader, entrepreneur a vu la nécessité de participer au développement de son pays en créant cette entreprise.

    le nom Sefray se définit t- il comme suit : Se= pour le nom, Fra= pour le prénom et Y= pour embellir les produits qu’offrent Sefray entreprise.

    Sefray entreprise vous offre déjà les produits tels que : Peinture Sefratex, Cola Marabout et autres. Sefratex c’est encore le nom de ce jeune. La création de cette entreprise vient dans un contexte très très spécial, en voyant à travers le pays les ménages qui laissent les sections communales pour se rendre dans les grandes villes pour trouver des peintures pour pouvoir peinturer leurs maisons. Il fallait trouver un moyen de permettre à ces gens de trouver avec faciliter les peintures pour embellir leurs maisons.
